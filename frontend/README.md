# DOCNEX Frontend

## Overview
This is the React-based frontend for the DOCNEX healthcare platform. It provides a modern, responsive interface for patients and doctors to manage appointments and medical records.

## Tech Stack
- **React 19**
- **TypeScript**
- **Vite**
- **Tailwind CSS** (if applicable, else standard CSS)
- **Axios** (for API communication)
- **React Router Dom** (for navigation)
- **Lucide React** (for icons)

## Getting Started

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

## API Configuration
The frontend connects to the backend API via the configuration in `src/api/client.ts`. By default, it points to `http://localhost:5000/api`.
